About NaijaFactCheck


NaijaFactCheck is a JavaScript-based web application designed to help users quickly identify fake or misleading WhatsApp messages and scams. It empowers users to verify suspicious messages directly in their browser.

The application uses a combination of keyword scanning, simulated AI analysis, and Hugging Face API-based AI checks to provide a reliable “fake news” evaluation. Analysis history is stored using localStorage so users can review past checks. This project was created as part of my CSE 310 Module 2 assignment to demonstrate applied programming skills, including software design, implementation, and testing.

🧠 How It Works

Messages are analyzed using two approaches:

  • 🔑 Keyword & Pattern Detection: We scan for common red-flag phrases like “Click here”, “FG palliative”, “free airtime”, and suspicious links (e.g., bit.ly, wa.me, tinyurl.com).
  • 🤖 AI Text Classification: We use a Hugging Face model (cardiffnlp/twitter-roberta-base-offensive) to classify messages as suspicious or not.
  • 🧪 Simulated AI Analysis: As a fallback or for faster checks, we use a fake API that mimics AI scoring based on local rules — no internet required.

This hybrid setup helps users detect scams quickly and reliably, even during showcase demos or in low-connectivity environments.
But
